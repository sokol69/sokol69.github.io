WebGL

Web-based Graphics Library - это контекст элемента canvas HTML, который обеспечивает API 3D графики без использования плагинов. WebGL исполняется как элемент HTML5 и поэтому является полноценной частью объектной модели документа браузера. За счёт использования низкоуровневых средств поддержки OpenGL, часть кода на WebGL может выполняться непосредственно на видеокартах.

Первая спецификация была выпущена 3 марта 2011 года. Современная версия 2.0 (несовместима с версией 1.0) доступна с 27 февраля 2017 года. Браузерная поддержка

ThreeJS

ThreeJS - библиотека JavaScript, используемая для создания и отображения анимированной 3D графики при разработке веб-приложений. Её работа основана на возможностях WebGL. ThreeJS выполняет всю низкоуровневую работу с WebGL и предоставляет набор классов с понятным интерфейсом.

Основные понятия

Рендер - сущность, отображающая на cnvas выбранную сцену с выбранной камеры.

Сцена - 3d пространство, в котором рапологаются источники света и элементы - меши.

Камера - точка обзора сцены с определенными характеристиками.

Источник света - элемент, создающий освещение сцены (м.б. рассеяным, точечным, направленным и т.д.).

Меш - элемент сцены, который состоит из геометрии и материала.

Геометрия - набор вершин, которые при отоброжении соединяются графическими примитивами.

Материал - способ отображения и внешний вид элемента.

Текстура - изображение, которое может использоватся в рамках материала для задания внешнего вида обьекта.
